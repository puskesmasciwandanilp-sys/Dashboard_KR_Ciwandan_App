  □ icon-72x72.png       (72x72 px)

  □ icon-128x128.png     (128x128 px)
  □ icon-144x144.png     (144x144 px)
  □ icon-152x152.png     (152x152 px)
  □ icon-384x384.png     (384x384 px)

MASKABLE ICONS (Android adaptive icons):
  □ icon-maskable-192x192.png
  □ icon-maskable-512x512.png
  
  Note: Maskable icons harus memiliki "safe zone" 
  di tengah (80% area) karena bisa di-crop.

APPLE/SAFARI:
  □ apple-touch-icon.png (180x180 px)
  □ safari-pinned-tab.svg (monochrome SVG)

FAVICON:
  □ favicon-16x16.png
  □ favicon-32x32.png
  □ favicon.ico (optional, multi-size)

─────────────────────────────────────────────────────

CARA MEMBUAT ICONS:
─────────────────────────────────────────────────────

OPSI 1: PWA Builder (Recommended)
  1. Buka https://www.pwabuilder.com/imageGenerator
  2. Upload logo minimal 512x512px
  3. Pilih "Generate"
  4. Download ZIP dan extract ke folder ini

OPSI 2: Real Favicon Generator
  1. Buka https://realfavicongenerator.net/
  2. Upload logo
  3. Customize settings
  4. Download package

OPSI 3: Figma / Canva
  1. Buat frame sesuai ukuran
  2. Desain icon dengan padding yang cukup
  3. Export sebagai PNG

─────────────────────────────────────────────────────

DESIGN GUIDELINES:
─────────────────────────────────────────────────────

✓ Logo/icon harus jelas pada ukuran kecil (72px)
✓ Gunakan background solid atau transparan
✓ Untuk maskable: logo di tengah 80% area
✓ Warna background: #0f172a (sesuai theme)
✓ Hindari detail yang terlalu kecil
